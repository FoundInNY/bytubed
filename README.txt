What is BYTubeD?
----------------
BYTubeD stands for "Bulk YouTube video Downloader". It is a Firefox extension that helps
in downloading YouTube videos in bulk.  It can be used to  download all/selected YouTube
videos on any page which has some YouTube video links, by clicking the BYTubeD menu item
in the Tools  menu or in the context menu (i.e., the right click menu). It makes the job
of downloading YouTube playlists easy.

Getting BYTubeD
---------------
To install BYTubeD please visit https://addons.mozilla.org/en-US/firefox/addon/bytubed/ 
and click on "Add to Firefox". 

To know more about BYTubeD, please visit http://bytubed.blogspot.com/
