New Development Version - 1.0.9a5 (Posted on: 30 August 2012)

This version (1.0.9a5) just fixes the issue with non-contiguous selection of videos in Firefox 15+. Please note that the earlier dev-version (1.0.9a4) has the following major feature additions:
  • Scan all tabs of the current window for YouTube links
  • Support for 3gp format
  • Fetch subtitles when they are available
Version 1.0.9a5 includes all these new feautures and fixes the issue of video selection. Please install 1.0.9a5 and explore the "Additional Features" tab. If you observe any unsual behaviour, please report it in the comments of this post.
