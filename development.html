Issues and Feature Suggestions

If you have any suggestions for new features or have observed any issues with BYTubeD, please let me know. If you have a GitHub account or if you don't mind creating one, please use the issues page; otherwise, you can post your suggestions or issue reports as comments below.

Source Code

If you want to get the source code of BYTubeD, you can download it in either zip or tar format. You can also clone the project with Git by running:

$ git clone git://github.com/msram/bytubed
