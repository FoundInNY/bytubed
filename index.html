Description

BYTubeD stands for "Bulk YouTube video Downloader" (or "Batch YouTube video Downloader"). It is a Firefox extension that helps users in downloading YouTube videos in bulk (i.e., downloading multiple YouTube videos at once in batch mode with a few clicks). It can be used to download all/selected YouTube videos on any page which has some YouTube video links. It makes the job of downloading YouTube playlists easy.

Features at a glance

  • Batch download YouTube videos on any page
  • Download YouTube Playlists (as a product of the ability to batch download)
  • Formats Supported: FLV, MP4, WebM (subject to availability)
  • Qualities Supported: 240p, 360p, 480p, 720p HD, 1080p HD, Original (subject to availability)
  • Can be used in combination with download managers like DownThemAll

For more details, please see the Help Page.
