Features at a glance

  • Batch download YouTube videos on any page
  • Download YouTube Playlists (as a by-product of the ability to batch download)
  • Formats Supported: FLV, MP4, WebM (subject to availability)
  • Qualities Supported: 240p, 360p, 480p, 720p HD, 1080p HD, Original (subject to availability)
  • Can be used in combination with download managers like DownThemAll

For more details, please see the FAQ page.
