A: New development version - 1.1.1a1
(Sep 19, 2012)

Features at a glance

  • Helps in batch downloading YouTube videos on any page
  • Can grab YouTube video links from: any web page, a local HTML or text file and system clipboard
  • Can be used to download YouTube Playlists (as a by-product of the ability to batch download)
  • Formats Supported: FLV, MP4, WebM, 3GP (subject to availability)
  • Qualities Supported: 144p, 240p, 360p, 480p, 720p HD, 1080p HD, Original (subject to availability)
  • Can download subtitles in the chosen language, if available.
  • Can be used in combination with download managers like DownThemAll

For more details, please see the FAQ page.

Dev-versions Renamed    (Posted on 04 March 2012)

Firefox is somehow treating 1.0.8a1 and 1.0.8a2 as lower versions than 1.0.8 and soon after installing 1.0.8a1 or 1.0.8a2, it is downgrading BYTubeD to 1.0.8. In order to make Firefox believe that these dev-versions are more recent, 1.0.8a1 has been renamed as 1.0.9a1 and 1.0.8a2 has been renamed as 1.0.9a2.
